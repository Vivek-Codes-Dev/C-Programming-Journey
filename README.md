# C Programming Journey 🚀

This repository tracks my progress as I learn the C programming language from scratch. My goal is to master the fundamentals of Computer Science to prepare for the Google STEP Internship.

## 🛠️ Concepts Mastered So Far

### 1. Basic Syntax & Operators
* Input/Output using `printf` and `scanf`.
* Arithmetic and Assignment operators.
* **Operator Precedence**: Understanding the order of operations in complex expressions.

### 2. Conditional Logic (Control Flow)
* **If-Else Ladders**: Implementing multi-stage decision making (e.g., Driving License eligibility).
* **Logical Operators**: Using `&&`, `||`, and `!` for complex conditions.
* **Ternary Operator**: Writing shorthand conditional code.

### 3. Real-World Logic Problems 🆕
* **Tax Calculator**: Calculating income tax based on different salary slabs.
* **Leap Year Checker**: Using mathematical logic to determine year types.
* **Character Analysis**: Using ASCII values to detect lowercase letters.
* **Max Finder**: Finding the greatest value among multiple inputs.
